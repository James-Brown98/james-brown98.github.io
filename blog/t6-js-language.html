Uses in JavaScript

Parentheses () - These can be used in a function to give a name to the input when you call the function. For example, function rectangleArea(width, height). This means that you call the function and the values you put in the parentheses are height and then width in that order.
They can also be used in a more obvious way, for use in mathematical equations, eg. (2+2) x 2 = 8.

Brackets [] - These are used in conjuction with arrays. So when you're storing an array you will enter it inside the brackets like so ["James", 18, "Christchurch"]. This info can then be accessed by using brackets, for example, to access "James" quickly you can use the array name and then [0] as that gets the fist entry. The final use for these brackets is again to access/locate information, this time it is in an object instead of an array. So, basically you take an object eg. dog, then you can access a property such as tail. Using bracket notation this would be done like dog["tail"].

Braces {} - These are used to wrap a function in what you want it to do. For example, function shift(){ return '' }, or for(var i = 0; i < sum.length; i++){ return sum[i]; }

Double and Single quotes "" '' - These can be used interchangably, their function is to identify a string. For example, you can print a string to console like so, console.log("Hello").
