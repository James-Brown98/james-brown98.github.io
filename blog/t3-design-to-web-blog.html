Responsive Websites

Responsive websites are important as they allow a good experience or all users over all devices. This is due to the fact that responsive web design is basically meaning that the page will change in regards to the size of the screen (In pixels)

Inside this responsive design we have mobile first design which allows the creator to make a webpage with the primary use being mobile phones. Therefore, there won't be a massive excess of code in CSS which will go unused.

Frameworks

Frameworks are essentially a base for your code. They have inserted certain rules that you can work from and they allow ease of use for the consumer.

Wireframes

A wireframe is a really basic design sketch of what you want your webpages to look like. It allows the devolper to have a rough idea as to what the layout will be and wear everything should be.
