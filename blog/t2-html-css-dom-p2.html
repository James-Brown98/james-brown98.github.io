Grid Based Design

Grid based design is exactly as it sounds, it's when a web page is split up into a grid. So there are rows which have columns of different sizes inside. Each row is usually split into 12.

Responsive Web Pages

The ever increasing popularity of responsive web design is due to the advancement of technology. With newer technology such as phones, tablets, notebooks, smart watches etc. it's increasing how accessible the internet is to us. Responsive Web Design means that the web page will respond to the screen size so that the page is easy to read and navigate on a wide range of devices.

Semantic Structure

This is basically writing your code as it should be laid out. For example, the most important title on the page should be headed h1 and then the navigation links should inside nav tags.
